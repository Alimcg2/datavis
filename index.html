About These Graphs

Above you will find posts with data visualizations of topics that I find interesting. Data comes from open source databases and APIs. The data I choose to visualize covers a wide range of topics, some might be political, some comical, and some more heavy. Some of the subject matter is sensitive and/or controversial, beware.

All of the code written for visualization has been done in R, using a wide array of packages. If you would like more specific details about the coding of each graph my github is available to the public.

Most, if not all, of the visualizations are interactive so please give the pages time to load. All graphs posted above were created by me for your interactive pleasure! Enjoy!
